# 部门任务管理看板 V2.0

一个苹果风格的部门任务看板管理系统，支持多账号协作、可拖拽泳道、甘特图视图、悬赏积分排行榜、明暗主题切换。

## 功能特性

- 🎯 **看板管理** — 可拖拽泳道卡片，横向/纵向双布局模式
- 📊 **甘特图** — 任务时间轴视图，支持拖动调整日期，按天/周/月查看（V2.0）
- 👥 **多用户协作** — 管理员/成员角色，支持自助注册
- 🏆 **积分排行** — 任务悬赏分值，完成后自动均分积分
- 🔍 **多维筛选** — 关键字搜索 + 优先级/认领者/创建者筛选
- ⏰ **过期提醒** — 任务过期醒目红色标识
- 🌗 **明暗主题** — 苹果风格 UI，支持 Dark/Light 切换
- 📋 **三视图** — 卡片视图、列表视图、甘特图视图自由切换
- ⚙️ **灵活配置** — 系统名称/图标/菜单/注册/可见性均可设置
- 🔧 **个性化列设置** — 列表/甘特图列设置、卡片字段设置，均按用户绑定（V2.0）
- 📂 **任务分类** — 独立分类维度，甘特图按分类折叠展开（V2.0）
- 🎨 **自定义优先级** — 优先级名称和颜色管理员可自定义（V2.0）

## 技术栈

- **后端**: Node.js + Express
- **数据库**: lowdb (JSON 文件)
- **前端**: Vue 3 (CDN) + SortableJS
- **认证**: JWT

## 快速开始

### 环境要求

- Node.js >= 16

### 安装与运行

```bash
# 克隆项目
git clone https://github.com/Luerlei/task.git
cd task

# 安装依赖
npm install

# 启动服务
node server.js
```

服务启动后访问 http://localhost:3000

### 默认管理员

- 用户名: `admin`
- 密码: `admin123`

> ⚠️ 首次登录后请立即修改默认密码

## 局域网部署

启动后，同一局域网内其他设备可通过 `http://<服务器IP>:3000` 访问。

## API 概览

| 模块 | 路径前缀 | 说明 |
|------|----------|------|
| 认证 | /api/auth | 登录、注册、修改密码 |

| 分类 | /api/categories | 任务分类管理（V2.0） |
| 优先级 | /api/priorities | 优先级管理（V2.0） |
| 排行 | /api/leaderboard | 积分排行榜 |
| 设置 | /api/settings | 系统设置 |
