# 部门任务管理看板系统 V1.0 — 设计书

基于 Node.js + Express + lowdb 后端和 Vue 3 CDN 单页前端，构建一个苹果风格的部门任务看板管理系统。支持多账号、权限控制、可拖拽泳道看板、悬赏积分排行榜、任务筛选与过期标识、明暗主题切换。

---

## 一、系统架构

```
task/
├── server.js              # Express 服务入口 + 全部 API 路由
├── package.json           # 项目依赖
├── db.json                # lowdb 数据文件（自动生成）
└── public/
    └── index.html         # 单文件前端（Vue 3 + CSS）
```

- 后端：Node.js + Express + lowdb，提供 RESTful API
- 前端：单个 index.html，通过 CDN 引入 Vue 3 和 SortableJS
- 通信：fetch API 调用后端接口
- 认证：JWT token，存于 localStorage，请求头 `Authorization: Bearer <token>`
- 部署：局域网内一台电脑运行 `node server.js`，其他人通过 IP:3000 访问

### 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端框架 | Vue 3 (CDN) | 无需构建工具，适合单文件应用 |
| 拖拽 | SortableJS (CDN) | 可靠的跨列拖拽体验 |
| 后端框架 | Express | 轻量 Node.js Web 框架 |
| 数据库 | lowdb v5 (ESM) | JSON 文件即数据库，零配置 |
| 认证 | jsonwebtoken | JWT 无状态认证，7天过期 |
| 密码 | bcryptjs | 密码哈希，保证基本安全 |
| ID生成 | uuid | 通用唯一标识符 |

│          │  └────────────────────────────────────────────┘ │

### 顶栏

- 左侧：胶囊式导航按钮（看板/设置），支持自定义名称和图标
- 右侧：主题切换（🌙/☀️）、用户信息、退出按钮
- 仅管理员可见「设置」导航

### 侧边栏

- 积分排行榜（始终可见）
- 分组筛选复选框

### 工具栏

- 搜索框 + 优先级/认领者/创建者下拉筛选
- 横纵切换按钮（位于卡片/列表按钮左侧）
- 卡片/列表视图切换

### 任务卡片

```
┌─────────────────────────────────┐
│ [玄] 任务标题  ⏰过期    💰 30  │  ← 优先级标签 + 过期徽章 + 分值
│ 任务描述文字截断显示...          │  ← 描述内容
│ 2025/05/20   👤👤              │  ← 截止日期 + 认领者头像
└─────────────────────────────────┘
```

- 优先级四阶：天（红）、地（橙）、玄（蓝）、黄（绿）
- 卡片左侧彩色边框表示优先级
- 过期任务：红色边框 + 背景渐变 + 「⏰过期」徽章 + 截止日期红色加粗
- 悬赏分值显示在卡片右上角
- 日期格式：YYYY/MM/DD

### 列表视图

- 表格形式展示，列：泳道、优先级、标题、描述、分值、截止日期、认领者、创建者、创建时间
- 过期日期以红色高亮显示
- 点击行打开任务详情

### 系统设置页面

- 标签页切换：「用户管理」|「分组管理」|「泳道设置」|「功能开关」
- 功能开关包含：
  - 允许注册开关
  - 任务可见性控制开关
  - 系统名称/图标设置
  - 菜单名称/图标设置

## 六、泳道布局

### 横向模式（默认）

- 泳道作为列从左到右并排展示
- 各泳道**等分窗口可用宽度**（`flex:1`），无横向滚动条
- 泳道高度铺满页面窗口高度
- 卡片宽度自动适配泳道宽度（`width:100%`），单列纵向排列
- 卡片超出时泳道内部纵向滚动

### 纵向模式

- 泳道从上到下堆叠展示
- 泳道高度等分窗口剩余空间（`flex:1`）
- 泳道宽度 `width:100%` 匹配窗口，无横向滚动
- 卡片使用固定宽度（260px），泳道内 flex-wrap 横向多列排列
- 卡片超出时泳道内部滚动

### 视图切换

- 卡片/列表视图切换按钮
- 横纵方向切换按钮（仅卡片视图下显示）
- 切换状态保存在 localStorage

## 七、搜索与筛选

- **关键字搜索**：匹配任务标题和描述
- **优先级筛选**：天/地/玄/黄，下拉选择
- **认领者筛选**：下拉选择用户
- **创建者筛选**：下拉选择用户
- **分组筛选**：侧边栏复选框，按用户所属分组过滤
- 所有筛选条件同时生效（AND 逻辑）
- 卡片视图和列表视图均受筛选影响

---

## 八、积分规则

- 每个任务创建时设定悬赏分值
- 任务完成后（移入最后一个泳道），积分**均分**给所有认领者
- 积分排行榜展示所有成员的累计积分排名
- 排行榜位于侧边栏，始终可见

---

## 九、任务可见性控制

- **开启时**：
  - 「待认领」泳道（第一个泳道）的任务所有人可见
  - 其他泳道仅显示**当前用户认领的任务**（管理员可见全部）
- **关闭时**：所有用户可看到全部泳道中的所有任务
- 管理员可在系统设置中开启/关闭

---

## 十、用户注册

- 登录页面提供注册入口
- 注册字段：用户名、密码、显示名称
- 注册后角色默认为 member，分组为空
- 管理员可在系统设置中开启/关闭注册功能
- 关闭时登录页不显示注册入口，API 返回拒绝

## 十一、设计决策

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 前端方案 | 单HTML + Vue 3 CDN | 降低部署复杂度，无需构建工具 |
| 认证方案 | JWT (7天过期) | 无状态，适合轻量后端 |
| 密码存储 | bcrypt 哈希 | 基本安全保证 |
| 拖拽方案 | SortableJS | 可靠的跨列拖拽 |
| 数据库 | lowdb v5 (JSON文件) | 零配置，适合小团队 |
| ES模块 | package.json `"type":"module"` | lowdb v5 要求 ESM |
| 初始管理员 | admin / admin123 | 首次登录后建议修改密码 |
| 主题 | 明暗双主题 | CSS变量切换，苹果风格 |
| 优先级 | 天地玄黄四阶 | 中国传统元素，辨识度高 |
